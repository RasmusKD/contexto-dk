# Contexto DK - Fondsstøtte & Fremtidsplaner

## Baggrund
- **Projekt:** Dansk version af Contexto ordspil
- **Teknologi:** DanNet semantisk kategorisering + Word2Vec embeddings
- **Udvikler:** Datamatiker (12 i PHP-version), nu PBA IT-sikkerhed studerende

## Fremtidige Udvidelser

### 1. Mobil App
- Native iOS/Android app
- Offline mode
- Push notifications for daily word
- Streak tracking

### 2. Multiplayer - Kahoot-stil (Klasseværelse)
**Koncept:** Host + deltagere i samme rum
- Lærer starter session, elever joiner med kode
- Samme målord for alle
- Live scoreboard
- Point for:
  - Flest korrekte gæt
  - Tætteste ord til målordet
  - Flest ord inden for kategori
  - Hurtigste til at finde målordet

**Sværhedsgrader:**
- Let: Kategori synlig
- Medium: Kategori skjult
- Svær: Ingen hints

### 3. Versus Mode (1v1)
**Koncept:** To spillere, samme målord
- Matchmaking eller invitation
- Hvem finder ordet på færrest gæt?
- Rating/ELO system
- Ranked seasons

**Varianter:**
- Speed round (tidsbegrænset)
- Best of 3/5
- Turnering mode

### 4. Kategori-baserede Modes
Fordi alle ord har kategorier kan vi lave:
- **Tema-dage:** Kun dyr, kun mad, kun kroppen
- **Kategori-gæt:** Find alle ord i en kategori
- **Kategori-kæde:** Find ord der forbinder to kategorier

### 5. Uddannelsesfeatures
- Ordforråds-tracking per elev
- Lærer-dashboard med statistik
- Integration med LMS (Moodle, etc.)
- Progression/curriculum mode

### 6. Tilgængelighed
- Ordblinde-venlig font
- Høj kontrast mode
- Skærmlæser support
- Synonym-hints
